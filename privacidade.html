Política de Privacidade

Última atualização: 29/04/2026

1. Coleta de Dados

Este aplicativo coleta apenas dados de anúncios do Facebook Ads necesarios para análise de desempenho de campanhas publicitárias. Os dados coletados incluem: impressões, cliques, gastos, alcance, conversões e métricas relacionadas.

2. Uso dos Dados

Os dados são utilizados exclusivamente para:

  • Dashboard de análise de performance de anúncios
  • Relatórios de métricas de campanhas
  • Visualização de resultados de marketing

3. Armazenamento

Os dados são armazenados localmente no dispositivo do usuário e não são compartilhados com terceiros.

4. Direitos do Usuário

O usuário pode solicitar a exclusão de todos os dados a qualquer momento através do formulário de exclusão.

5. Contato

Para dúvidas sobre privacidade, entre em contato através do email fornecido no app.
