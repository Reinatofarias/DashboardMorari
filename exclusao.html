Solicitação de Exclusão de Dados

Última atualização: 29/04/2026

Como solicitar a exclusão

Para solicitar a exclusão de todos os dados pessoais armazenados por este aplicativo, siga os passos abaixo:

Método 1: Exclusão Automática

Para excluir todos os dados armazenados localmente:

  • Exclua a pasta "data" no diretório do aplicativo
  • Ou.delete("facebook_ads_latest.json") arquivo

Método 2: Contato por Email

Envie um email para o desenvolvedor solicitando a exclusão com o assunto "Exclusão de Dados - PowerBI META"

Prazo de Resposta

Todas as solicitações serão processadas em até 30 dias.

Dados Excluídos

Serão excluídos:

  • Arquivos de dados CSV, JSON e XLSX
  • Cache local
  • Tokens de acesso (se armazenados)
